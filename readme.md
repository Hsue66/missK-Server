# Miss_K App v0.0.0
The Easiest way to know micro dust info

## Super Easy Install

### install MongoDB
  - [Mongodb Download Link](https://www.mongodb.com/download-center?jmp=nav#community)

### Run MongoDB Server
```bash
$ mongod
```

### Clone Repository
```bash
$ git clone https://github.com/produce105/missK-Server
```

### First of all, install node_modules
```bash
$ npm install
```

### Run server
```bash
$ npm start
```
